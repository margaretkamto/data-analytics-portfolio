# Data Note

The raw datasets are not included in this repository because of file size and licensing considerations.

This project used two open-access datasets:

1. **METABRIC Breast Cancer Dataset**  
Used for the unsupervised learning task.
2. **Bank Marketing Dataset from the UCI Machine Learning Repository**  
Used for the regression and classification tasks.

The METABRIC dataset was used to explore patterns in breast cancer-related clinical and genetic data through unsupervised learning methods.

The Bank Marketing dataset was used to build supervised learning models for regression and classification analysis.

This repository only includes the notebooks, final report, and supporting documentation.
