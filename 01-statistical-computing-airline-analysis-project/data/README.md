# Data Note

The raw airline data files are not included in this repository because the original dataset is very large.

The data used in this project comes from the 2009 ASA Statistical Computing and Graphics Data Expo, hosted on Harvard Dataverse.

Data source: https://doi.org/10.7910/DVN/HG7NV7

## Data Used

The original analysis used flight records from 1997 to 2006.

The project also used supplementary data, including:

- Plane data
- Airport data

These files were used to add information such as plane age and airport coordinates.

## Why the data is not uploaded

The full airline dataset is too large to store directly in this GitHub repository. Because of that, this repository only includes the notebooks, R Markdown files, reports, and documentation.s
