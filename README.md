# Data Analytics Portfolio

This repository contains selected academic projects from different university modules, mainly focused on data analytics, machine learning, statistical computing, and business analysis.

The projects include coding-based analysis using Python and R, machine learning model development, and dashboard-based business analysis using Tableau. I am using this repository to organize my technical and analytical work while preparing for internship and entry-level job applications.

## Projects

### 1. Statistical Computing and Airline Delay Analysis

This project was completed for a Programming for Data Science module. It includes two parts: a simulation task using the random walk Metropolis algorithm, and an airline delay analysis using US flight data from 1997 to 2006.

The airline analysis looks at delay patterns by day and time, the relationship between plane age and delay, and factors related to flight diversion.

**Tools:** Python, R, Jupyter Notebook, R Markdown  
**Methods:** MCMC simulation, random walk Metropolis, exploratory data analysis, logistic regression  
**Folder:** `01-statistical-computing-airline-analysis-project`

### 2. Machine Learning Project

This project was completed for a Machine Learning module. It applies unsupervised learning, regression, and classification using real-world datasets.

The unsupervised learning section uses the METABRIC breast cancer dataset for PCA and clustering. The regression and classification sections use the Bank Marketing dataset to analyse customer call duration and term deposit subscription.

**Tools:** Python, Jupyter Notebook, pandas, NumPy, scikit-learn, XGBoost  
**Methods:** PCA, K-Means clustering, hierarchical clustering, regression, classification, model evaluation  
**Folder:** `02-machine-learning-analysis-project`

### 3. Tableau Business Analysis Project

This project was completed for a Business Analytics, Applied Modelling & Prediction module. It uses Tableau dashboards to analyse sales, customer behaviour, shipping performance, profit margins, and profit forecasting based on four years of sales data.

The report focuses on interpreting dashboard results and translating the findings into business recommendations for senior management.

**Tools:** Tableau, Excel, dashboard analysis  
**Methods:** sales analysis, customer analysis, shipping analysis, profit analysis, forecasting, business recommendations  
**Folder:** `03-tableau-business-analysis-project`
